
Ici, il devrait y avoir le binaire de CSDM, mais il est trop lourd.

Pour le récupérer, il suffit d’aller sur la page des releases :
[https://github.com/akiver/cs-demo-manager/releases](https://github.com/akiver/cs-demo-manager/releases)

Télécharge l’AppImage Linux correspondant (par exemple `CS-Demo-Manager-3.18.4.AppImage`).

Sous Arch Linux :

1. Rendre le fichier exécutable :

```bash
chmod +x ~/Downloads/CS-Demo-Manager-3.18.4.AppImage
```

2. Extraire le contenu de l’AppImage :

```bash
~/Downloads/CS-Demo-Manager-3.18.4.AppImage --appimage-extract
```
